# SomePackage

Boilerplate for Python (3) projects.

--

## Installing

```sh
# you need setuptools
$ python setup.py install

# there is a default executable with logging
# and argument parsing. See `somepackage/__main__.py`
$ spackage
INFO:root:Hello, World!
```

## Testing

```sh
# like nose?
$ python3 -m "nose"
```
